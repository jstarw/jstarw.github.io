The code is seperated into three javascript files:

  • main.js: Main running function of the application. It calls up interactive.js and util.js
  • interactive.js: Contains functions dealing with interactivity, such as tooltip
  • util.js: Contains utility functions (create graph, controller functionality, etc.)
